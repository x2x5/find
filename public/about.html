淘顶网

像逛淘宝一样逛顶会论文。

这是什么？

淘顶网 是一个 AI 顶会论文浏览器。覆盖 MLCVAI 三大领域共 9 个顶会。

所有论文随机打乱展示——你每次搜索，看到的结果顺序都不一样。就像在淘宝上逛，你不知道下一页会出现什么。

为什么叫淘顶网？

淘 = 淘宝的淘。我们希望找论文像逛淘宝一样轻松有趣。

顶 = 顶会。只收录顶级会议，不收录水会。

你可以把喜欢的论文加入购物车，最后一次性复制导出。还能看到最有缘分的论文——搜索时随机推荐一篇。

数据从哪来？

论文标题数据来自两个渠道：

🔹 DBLP — 计算机科学文献数据库，手工提取整理

🔹 Paper Copilot — 部分数据从该网站获取

数据存放在 papers/<conference>/<year>.txt，每行一个标题。运行脚本生成 JSON 供前端读取。

"最有缘分"是什么？

每次输入关键词搜索后，系统会随机从结果中挑一篇论文展示在顶部——

就像茫茫论文海中，最有缘分的那一篇主动出现在你面前。

点击标题直接加入购物车。

不输入关键词时，不显示——缘分要自己去寻找。

怎么部署到自己的电脑？

第一步：安装 Node.js

打开终端，用包管理器安装：

# macOS
brew install node

# 或从官网下载: https://nodejs.org

验证安装：node -v && npm -v

第二步：克隆项目

git clone https://github.com/x2x5/find.git
cd find

第三步：安装依赖并启动

npm install
npm run dev

浏览器打开 http://localhost:5173 即可。

怎么更新论文数据？

在 papers/ 目录下找到对应会议和年份的 .txt 文件，添加新标题（一行一个）。

然后运行：

cd scripts
npx tsx gen-data.ts

刷新网页即可看到新数据。
