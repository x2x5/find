GitHub Token 怎么弄

这个页面是给购物车右下角问号按钮用的。看完可以直接回主页继续用。

为什么要填 Token

购物车会帮你查每篇论文可能对应的 GitHub 仓库。这个查询走的是 GitHub API，不带 Token 很容易撞到限流。

怎么创建

  1. 打开 GitHub Token 设置页
  2. 点 Generate new token
  3. 随便起个名字，比如 find-local
  4. 如果只是这个项目本地用，通常不需要额外仓库权限
  5. 生成后把 Token 复制出来，填进购物车里的钥匙按钮

保存到哪里

只保存到你当前浏览器的 localStorage。不会自动上传到 GitHub，也不会写进这个仓库代码里。

为什么填了还可能失败

  • Token 本身失效了
  • GitHub Search API 仍然有频率限制
  • 论文标题确实没搜到对应仓库
如果你之前在聊天里公开发过 Token，那枚 Token 就应该立刻废掉，重新生成新的。
