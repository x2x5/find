淘鼎 TaoDing

三大顶会论文检索 — 快速浏览、筛选、收藏 AI 顶会论文标题。

设计理念

三大顶会 = 三个领域 × 每组三个会议：

ML NeurIPS · ICML · ICLR
CV CVPR · ECCV · ICCV
AI AAAI · MM · IJCAI

所有论文随机打乱，每页更像「淘」的感觉——你不知道下一页会看到什么。顶部的"运气最佳"随机展示一篇，点标题直接加入购物车。

功能一览

  • 实时搜索 — 边打字边过滤，搜索结果数实时显示
  • 年份筛选 — 可调范围，一键重置最近两年
  • 领域分布 — 柱状图实时显示各会议论文数量，点击切换筛选
  • 年份分布 — 按年份查看论文数量变化
  • 会议时间轴 — 投稿/录用日期可视化，可折叠
  • 运气最佳 — 每次搜索随机推荐一篇
  • 购物车 — 收藏论文，一键复制导出
  • 深色模式 / 中英切换

本地部署

cd v2
npm install
npm run dev

浏览器打开 http://localhost:5173

生成生产版本：npm run build，部署 dist/ 目录即可。

数据更新

论文数据存放在 papers/<conference>/<year>.txt，每行一个标题。

更新后运行：

cd v2/scripts
npx tsx gen-data.ts

技术栈

  • React 18 + TypeScript
  • Tailwind CSS 3.4
  • Vite 6
  • Lucide React (图标)
